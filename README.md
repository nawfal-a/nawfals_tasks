Globally listenable, Concurrent tasks.

## Features

* Configurable maximum parallel tasks.
* Auto cancellation of repeated task.
* Task debounce.
* Configurable auto retry.
* Can listen to task anywhere in the app.
* Highly customizable tasks listeners.
* Support for task progress.
* Support task data attachment.
* Tasks can be tagged.
